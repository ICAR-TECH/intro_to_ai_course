# Introduction to Artificial Intelligence
## Dr. Samarth Godara, ICAR–IASRI, New Delhi, India

## Course Page: 

This course aims to familiarize students with the core principles of artificial intelligence and introduce them to a wide range of problem types that can be effectively addressed using AI methodologies.

The practical sessions have been thoughtfully designed to run on low-configuration computers, ensuring broad accessibility and enabling students to gain hands-on experience through straightforward coding exercises.

All course materials—including video lectures, presentations, source codes, and datasets—are freely available to students.
